# ODK-means
Official repository with (almost) all the procedures shown in "ODK-means: a simultaneous approach to clustering and outlier detection".

All the functions needed to reproduce exactly all the results in the paper are available. Some results have been directly loaded as .R files (e.g. Figure 4 and Figure 5). 

All functions have been "commented and packaged" by AI (my versions are much more "spartan", as you would imagine). For this reason, if you notice any problem you can contact me directly and I will send you the "working" function or the solution (hopefully) to your problem. 

Currently (11/03/2025) everything works just fine.

Functions are NOT optimized. ODK-means is a statistical procedure, and I did not spend time in the optimization. If required, I will do it.

The data generation procedure has some exceptions which have not been managed (it was not required for the paper).
